Introduction

Java is a versatile, object-oriented programming language widely used for developing desktop, web, and mobile applications. Created in 1995 by Sun Microsystems, Java is known for its "write once, run anywhere" philosophy, which allows programs to run on any device with a Java Virtual Machine (JVM).

Its syntax is similar to C++ but includes simplified features to reduce common programming errors. Java's extensive libraries and frameworks make it a go-to choice for developers working on enterprise-level projects. With its focus on portability, security, and performance, Java remains a cornerstone in the software development industry.

What you should already know

To understand this technical document, you must know some of the following:

  • Basic Programming Concepts: Understanding variables, loops, and conditional statements is essential before learning Java.
  • Familiarity with Other Languages: Experience with languages like Python, C, or C++ can ease your transition into Java.
  • Object-Oriented Programming (OOP): Knowledge of OOP principles such as classes, objects, inheritance, and polymorphism is highly beneficial.
JavaScript and Java

Although their names suggest a connection, JavaScript and Java are fundamentally different programming languages designed for distinct purposes. Java is a statically typed, object-oriented language used primarily for building large-scale applications, while JavaScript is a dynamically typed scripting language mainly used for web development.

Variables

In Java, variables are used to store data that can be manipulated during program execution. Java requires developers to declare a variable's type, such as int, double, or String, ensuring type safety and reducing runtime errors.

Java supports different scopes for variables, including local, instance, and static, which determine their accessibility. Understanding how to declare and use variables effectively is crucial for writing efficient and maintainable Java code.

Declaring variables

In Java, variables are declared with a specific data type, ensuring strong type safety and reducing runtime errors.

  • Basic Declaration: A variable is declared by specifying its type followed by its name, e.g., int x = 42;, which creates an integer variable initialized to 42.
    • int x = 42; double pi = 3.14; String message = "Hello, Java!";
  • Local Variables: Variables declared inside a method or block are local and accessible only within that scope.
  • Instance Variables: These are declared inside a class but outside any method, making them accessible to all methods within the class. public void displayMessage() { int localNumber = 10; System.out.println("Local number: " + localNumber); }
  • Static Variables: Declared with the static keyword, these variables belong to the class rather than any specific instance, making them shared across all objects. Java's strict variable declaration rules enforce reliability, ensuring code is robust and less prone to unintended behavior.
if...else statement

The if...else statement in Java is a fundamental control structure used for decision-making. It evaluates a boolean expression and executes one block of code if the condition is true, or another block if the condition is false.

For example, if (score >= 50) { System.out.println("Pass"); } else { System.out.println("Fail"); } determines whether a score meets a passing threshold. Java also supports else if clauses for handling multiple conditions, making it a versatile tool for flow control. Mastering if...else statements is essential for writing dynamic and responsive programs.
